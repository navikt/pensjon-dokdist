== Grensesnitt
REST tjeneste for å ferdigstille distribuere journalpost.

Se swagger dokumentasjon:  https://pensjon-dokdist.dev-fss.nais.io/swagger-ui.html

=== Input

|===
|Aributt |Type |Påkrevd |Beskrivelse

|id
|String
|Ja
|Journalpost som skal distribueres

|status
|String
|Ja
|Status på journalposten, f.eks. FL eller FS.

|distribusjonstype
|Enum: VEDTAK, VIKTIG, ANNET
|Nei
|Kreves ved distribusjon av fritekst-brev, ellers ignoreres den.

|===

=== Output
BestillingsId
